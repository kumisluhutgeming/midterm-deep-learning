# midterm-deep-learning
Repository ini berisi implementasi lengkap dari awal hingga akhir untuk tugas Midterm pada kelas Machine Learning dan Deep Learning. Proses yang dikerjakan mencakup pembersihan data, penanganan data yang hilang dan nilai ekstrem, pembuatan fitur, pembangunan model, proses pelatihan, evaluasi, serta penjelasan terhadap hasil yang diperoleh.

Pada seluruh tugas yang diberikan, yaitu fraud detection, regresi, dan clustering, pipeline dibuat untuk membandingkan berbagai algoritma, melihat performa model, dan menarik insight yang relevan dari dataset (meskipun pada akhirnya hanya tugas regresi yang dapat diselesaikan sepenuhnya). Pengerjaan ini menunjukkan bagaimana teknik machine learning dan deep learning dapat diterapkan secara nyata pada dataset berukuran besar.
