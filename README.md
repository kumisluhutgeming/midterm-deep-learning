# midterm-deep-learning
Repository ini berisi implementasi lengkap dari awal hingga akhir untuk tugas Midterm pada kelas Machine Learning dan Deep Learning. Proses yang dikerjakan mencakup pembersihan data, penanganan data yang hilang dan nilai ekstrem, pembuatan fitur, pembangunan model, proses pelatihan, evaluasi, serta penjelasan terhadap hasil yang diperoleh.

Pada seluruh tugas yang diberikan, yaitu fraud detection, regresi, dan clustering, pipeline dibuat untuk membandingkan berbagai algoritma, melihat performa model, dan menarik insight yang relevan dari dataset (meskipun pada akhirnya hanya tugas regresi dan tugas prediksi yang benar-benar dapat diselesaikan). Tugas prediksi berjalan cukup optimal, bahkan dapat dijalankan secara lokal dengan TensorFlow berbasis CPU tanpa kendala berarti, sedangkan tugas regresi justru menunjukkan performa yang sangat lambat ketika dijalankan di Google Cloud.

Implementasi ini menunjukkan bagaimana teknik machine learning dan deep learning dapat diterapkan secara nyata pada dataset berukuran besar sekaligus memberi gambaran bahwa tidak semua model memperlakukan hardware dengan tingkat kesabaran yang sama.
